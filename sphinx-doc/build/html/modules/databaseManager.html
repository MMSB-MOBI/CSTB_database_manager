addGenome(fasta: str, name: str, taxid: int = None, gcf: str = None, acc: str = None)¶

Take informations about genome and corresponding taxon and insert them in databases.

Parameters

  • fasta (path) – Path to fasta file

  • name (str) – Taxon name

  • taxid (int) – Taxid if available

  • gcf (str) – GCF accession for assembly if available

  • acc (str) – accession number if available

bind(genome, taxon)¶

Make the link between genome and taxon. Add genome uuid in taxon entry and taxon uuid in genome entry with consistency checks.

Parameters

  • genome (CSTB_database_manager.genome.GenomeEntity) – GenomeEntity that represents a genome couch entry

  • taxon (CSTB_database_manager.genome.TaxonEntity) – TaxonEntity that represents a taxon couch entry

Raises

  • error.VersionError – Raise if genome already exists as an older version of taxon.

  • error.LinkError – Raise if links already exists and don’t correspond.

createTree()¶

Create taxonomic tree from taxon database. Will automatically search taxon, create tree and store tree into database. Use engine.taxonomic_tree to create and format tree.
